# GitHub User Finder

**Автор:** иван иванов

## Описание
Приложение для быстрого поиска профилей пользователей в GitHub. Программа позволяет просматривать базовую информацию о пользователе и сохранять ссылки на интересные профили в список "Избранное".

## Как использовать API
Приложение использует **GitHub REST API** (публичная конечная точка `https://api.github.com/users/{username}`). 
* Регистрация ключа API для простых запросов не требуется (действуют лимиты GitHub на количество запросов в час).
* Для получения данных отправляется GET-запрос, ответ обрабатывается в формате JSON.

## Инструкция по запуску
1. Установите необходимые библиотеки: `pip install requests`.
2. Запустите файл: `python main.py`.
3. Введите логин пользователя GitHub в поле поиска и нажмите "Найти".

## Примеры использования
* **Тест 1 (Поиск):** Введите `vinta` (создатель awesome-python). Приложение должно показать имя и биографию.
* **Тест 2 (Избранное):** После поиска нажмите "В избранное". Пользователь должен мгновенно появиться в нижнем списке.
* **Тест 3 (Ошибка):** Оставьте поле пустым. Программа выдаст предупреждение о некорректном вводе.
